## 📌 담당 기능
#### 펫시터 지원
- 자격증 사진 첨부
- 자기소개서 작성
- 희망 일당 작성


#### 펫시터 목록
- 펫시터 승인 완료 후 자신의 페이지 작성하면 생성
- 해당 박스에 프사, 제목, 내용, 별점, 리뷰 수 노출


#### 펫시터 목록 추가 페이지
- 제목, 자기소개서, 프사, 배경화면 등록


#### 펫시터 상세페이지
- 배경화면, 프사, 자격증 여러 파일들 노출
- 펫시터가 작성한 제목, 내용
- 리뷰 수, 평점
- 자신의 게시글이면 수정하기 버튼이 보임
- 달력으로 해당 날짜 예약기능 구현
- 펫시터 리뷰 노출, 리뷰를 쓴 회원이면 수정하기 버튼이 보임
